# 🚀 Student Team Management System

## 📌 Project Overview

The **Student Team Management System** is a web-based application designed to manage student teams efficiently. It allows users to create teams, manage members, assign roles, and track team activities in an organized way.

---

## 🎯 Features

* 👥 Create and manage student teams
* ➕ Add, update, and remove team members
* 🔐 User authentication (Login/Signup)
* 📂 Organized backend with REST APIs
* 🌐 Responsive frontend interface
* 📊 Efficient data handling using database

---

## 🛠 Tech Stack

**Frontend:**

* React.js
* HTML, CSS, JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

## ⚙️ Installation & Setup

### 🔹 1. Clone the repository

```bash
git clone https://github.com/AkashNaidu2004/student-team-management.git
cd student-team-management
```

---

### 🔹 2. Install dependencies

#### Backend:

```bash
cd backend
npm install
```

#### Frontend:

```bash
cd ../frontend
npm install
```

---

### 🔹 3. Run the project

#### Start Backend:

```bash
cd backend
node server.js
```

#### Start Frontend:

```bash
cd frontend
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file inside `backend/` and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
```
